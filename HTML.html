HTML Nedir?

HTML, web sayfalarının iskeletini oluşturan bir işaretleme dilidir. İnternet üzerindeki tüm web siteleri, HTML kullanılarak yapılandırılır.

HTML'in Tarihçesi

HTML, 1990 yılında Tim Berners-Lee tarafından geliştirilmiştir. İlk versiyonu oldukça basit olan HTML, günümüzde sürekli gelişerek modern web uygulamalarının temelini oluşturuyor.

HTML Özellikleri

  • Web sayfalarının yapısını oluşturur
  • Metin, görseller, videolar ve diğer içerikleri destekler
  • Çeşitli tarayıcılarla uyumludur
  • Bağlantılar ve formlar gibi etkileşimli öğeler sunar
  • CSS ve JavaScript ile entegre çalışabilir
  • Hızlı ve kolay öğrenilebilir

HTML Örneği

Aşağıda temel bir HTML kullanım örneği bulunmaktadır:



<!DOCTYPE html>

<html>

  <head>

    <title>Merhaba Dünya</title>

  </head>

  <body>

    <h1>Merhaba Dünya!</h1>

    <p>Bu bir HTML örneğidir.</p>

  </body>

</html>
