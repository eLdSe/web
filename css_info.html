CSS Nedir?

CSS, Cascading Style Sheets'in kısaltmasıdır. Web sayfalarının stilini ve düzenini tanımlamak için kullanılan bir dildir. CSS, HTML ile birlikte çalışarak web sayfalarının görünümünü düzenler ve daha estetik bir tasarım sağlar.

CSS'in Tarihçesi

CSS, 1996 yılında W3C tarafından standartlaştırıldı. İlk sürümünden bu yana gelişerek CSS2 ve CSS3'e ulaştı. CSS3, web tasarımında devrim yaratan birçok yeni özellik ekledi.

CSS Özellikleri

  • Web sayfası tasarımını özelleştirir
  • Renk, yazı tipi, arka plan ve düzen gibi görsel özellikleri tanımlar
  • Responsive (duyarlı) tasarım sağlar
  • HTML ve JavaScript ile sorunsuz çalışır
  • Hızlı yükleme için dış stil dosyaları oluşturur
  • Animasyon ve geçiş efektlerini destekler

CSS Örneği

Aşağıda temel bir CSS kullanım örneği bulunmaktadır:


/* CSS Dosyası: style.css */
body {
    background-color: #f4f4f4; 
    color: #333;
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
}
h1 {
    color: #2196F3;
    text-align: center;
}
