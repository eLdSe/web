JavaScript Nedir?

JavaScript, web sayfalarına interaktif özellikler eklemek için kullanılan bir programlama dilidir. HTML ve CSS ile birlikte çalışarak web uygulamalarının dinamikliğini sağlar.

JavaScript'in Tarihçesi

JavaScript, 1995 yılında Brendan Eich tarafından geliştirildi. İlk olarak "Mocha" olarak adlandırılan dil, zamanla "LiveScript" ve ardından "JavaScript" adını aldı. Bugün, web geliştirmede en yaygın kullanılan dillerden biridir.

JavaScript Özellikleri

  • Web sayfalarını dinamik hale getirir
  • Olay yönetimini destekler (örneğin, fare tıklamaları ve klavye girişleri)
  • HTML ve CSS ile sorunsuz entegrasyon sağlar
  • API'lerle etkileşim kurabilir
  • Veri işleme ve doğrulama için kullanılır
  • Animasyon ve görsel efektler oluşturabilir

JavaScript Örneği

Aşağıda temel bir JavaScript kullanım örneği bulunmaktadır:


// Basit bir "Merhaba Dünya" örneği
console.log("Merhaba Dünya!");

// HTML elementi ile etkileşim
const button = document.querySelector('button');
button.addEventListener('click', function() {
    alert('Butona tıklandı!');
});
